# STM32-HMI_MP3
该项目是为别人做的毕业设计，是利用串口屏和语音播报模块做的MP3播放器
## 功能简介
用户可以直接触摸屏幕按钮进行控制，上一曲 下一曲 暂停 播放 音量加 音量减都已经实现
## 所需硬件
- 淘晶驰X5触摸串口屏（别的系列也可，换汤不换药）
- STM32F103C8T6单片机
- 语音播报模块dysv5w
## 工程结构
HMI_test是开发串口屏的工程
另一个是STM32工程
## 引脚连接
### 串口屏部分
- RXD--PB11
- TXD--PB10
- VCC--5V
### 语音播报模块
- 5V---单片机5V
- TX----PA10
- RX----PA9
